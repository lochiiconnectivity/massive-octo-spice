### Configuring Bind9
1. modify /etc/bind/named.conf.options to point at public-dns

    ```
    options {
        ...
        forward only;  
        forwarders {  
            8.8.8.8;  
            8.8.4.4;
        };
        ...
    };
    ```

1. verify /etc/bind/named.conf.local

    ```
    // bypass any forwarders

    zone "cymru.com" {
        forward only;
        type forward;
        forwarders { };
    };

    zone "zen.spamhaus.org" {
        forward only;
        type forward;
        forwarders { };
    };

    zone "dbl.spamhaus.org" {
        forward only;
        type forward;
        forwarders { };
    };
    ```
1. reload bind
1. verify bind is working properly

    ```
    $ dig ns1.google.com
    ```
